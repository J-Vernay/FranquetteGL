Global EGL status modification

In EGL, each thread stores the status of the last EGL function call.

Additionally, in EGL_KHR_debug extension, the user can register a callback to be called on certain events. Using these functions ensures that everything is taken care of.

enum FEGL_DebugTypeMask

Bitset representing the interest of user in the different message types.

Enumerator

Internal EGL failures (BAD_ALLOC, CONTEXT_LOST)

fegl_DEBUG_ERROR_BIT 

Input and bad match errors (BAD_PARAMETER...)

fegl_DEBUG_WARN_BIT 

Warnings, deprecated or inefficient operation.

fegl_DEBUG_INFO_BIT
